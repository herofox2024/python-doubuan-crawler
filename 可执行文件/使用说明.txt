# 豆瓣书评爬虫工具 - 独立版本

## 🎉 打包完成！

### 📦 生成文件
- **豆瓣书评爬虫.exe** (39.2 MB) - 独立可执行文件
- 包含所有必要依赖，无需安装Python环境

### 🚀 使用方法

#### 1. 启动程序
- 双击 `豆瓣书评爬虫.exe` 启动GUI界面
- 首次运行会在程序目录创建数据库文件

#### 2. 配置Cookie
- 在GUI界面中输入豆瓣用户名和Cookie
- Cookie获取方法：
  1. 浏览器登录豆瓣 → 访问收藏页面
  2. F12开发者工具 → Network标签页  
  3. 刷新页面 → 找到页面请求 → 复制Cookie

#### 3. 开始爬取
- 设置最大页数（建议先测试5-10页）
- 点击"开始爬取"按钮
- 实时查看进度和日志信息

#### 4. 导出数据
- 爬取完成后点击"导出HTML"
- 选择保存位置和文件名
- 生成交互式HTML报告

### 📂 生成文件说明
- `douban_books.db` - 数据库文件，存储所有爬取数据
- `gui_config.txt` - 配置文件，保存用户名和Cookie
- `{用户名}_豆瓣书评.html` - 导出的HTML报告

### ⚠️ 注意事项
1. **首次使用**：程序会在同级目录创建数据库文件
2. **Cookie时效**：Cookie会过期，需要定期更新
3. **网络连接**：确保网络连接稳定
4. **爬取频率**：建议适当控制爬取速度

### 📋 系统要求
- Windows 7/8/10/11
- 无需安装Python或其他依赖
- 建议内存 2GB 以上
